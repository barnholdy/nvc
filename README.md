# nvc

## Vue CLI 3 setup
```
npm install -g @vue/cli
npm install -g @vue/cli-service-global
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploys to GitHub Pages
```
npm run deploy
```
